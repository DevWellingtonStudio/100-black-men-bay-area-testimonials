An easy to use the plugin that allows users to add Testimonials to the sidebar, a widget, or to embed testimonials into a Page or Post using the shortcode. [testimonial-slider]




<strong>How To Use Shortcodes? </strong>
basic Shortcode
<code>
[testimonial-slider]
</code>
<strong>Note: </strong>
Copy the above shortcode to the page, post or widget section to display the testimonial in the website.
